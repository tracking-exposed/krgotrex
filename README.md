## CONTRIBUTION

### Front-end development

1. Fork this repo. [Here is a How-to](https://help.github.com/articles/fork-a-repo/)
2. `cd` into the project
3. `npm install`
4. `npm start`

Whenever you do *style* change you'll have to manually reload the page.
